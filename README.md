# StudyGrid.ver.1.9
StudyGrid is a web app that lets you drag and drop your AS‑Level chapters onto a calendar, track your exam countdown, and get AI help for any topic using a built‑in StudyGrid AI assistant.

Features
Calendar scheduler for all AS subjects (Math, Chemistry, Physics, Business, English).

Drag and drop chapters from a sidebar onto specific days.

Drag tasks between days or back to the sidebar to unschedule.

Collapsible subject sections with all chapters and subtopics.

StudyGrid AI chat page with:

Perplexity‑style interface.

File and image upload (or paste from clipboard).

“Learn step by step” plans for every chapter.

Textbook‑style maths formatting (×, ÷, roots, powers).

Exam countdown page with live timers for all 2026 exam papers.

LocalStorage persistence so your schedule and settings stay saved in the browser.

Tech stack
HTML5, modern CSS (flexbox, grid).

Vanilla JavaScript (no frameworks).

Google Gemini API for AI features.

Getting started
Download or clone this repository.

Open study-scheduler.html (or index.html) in any modern browser.

(Optional) Host it online with GitHub Pages or any static hosting provider.

Usage
Expand a subject in the sidebar and drag any chapter onto a day in the calendar.

Click a task’s × icon to remove it, or drag it back to the sidebar to unschedule.

Click StudyGrid AI to open the full AI page and chat about any topic or file.

Use the Learn step by step menu to generate a structured learning path for a chapter.

Click Exams to see a live countdown for each exam and how many days are left.
